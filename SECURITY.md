# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: arielfrischer@gmail.com

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 7 days
- **Resolution target**: Within 30 days (depending on severity)

### What to expect

1. You'll receive an acknowledgment of your report
2. We'll investigate and keep you updated on progress
3. Once fixed, we'll publicly disclose the issue (crediting you, unless you prefer anonymity)

Thank you for helping keep this project secure.
